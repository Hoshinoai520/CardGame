## 这版原型表达什么

- 白底战斗桌面：贴近视频里的功能框说明稿，而不是最终美术稿。
- 黄色功能框：费用、牌堆、额外效果等资源和承载区。
- 黑色组件：备战角色卡背、切换门、展开手牌竹简。
- 紫色箭头：回合结束与行动权切换。
- 右侧区域说明：点击画面编号即可看到每个框架的产品含义。

## 给 UI 设计师的重点

这份原型主要用于解释布局和信息层级。后续做视觉稿时，可以在不改变区域含义的前提下替换材质、角色立绘、卡牌背面、动效和按钮风格。
